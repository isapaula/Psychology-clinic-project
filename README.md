# Sistema para universidades com curso de psicologia
O objetivo desse sistema é proporcionar à comunidade a oportunidade de se consultar gratuitamente com estudantes do curso de psicologia.  A comunidade realiza o cadastro e seleciona o tipo de atendimento (presencial/híbrido), como a queixa (ansiedade, depressão, TDAH etc.). Após a comunidade fazer a solicitação de atendimento, ela estará na etapa de triagem.
Após o cadastro aprovado, o aluno mais indicado poderá realizar a consulta com ou sem o auxílio do professor, a depender do tipo da queixa.
